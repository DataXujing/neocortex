MNIST multi-layer perceptron

This demonstrates a 3-layer MLP with ReLU activations and dropout, culminating in a 10-class softmax function which predicts the digit represented in a given 28x28 image. Offline, the architecture and weights of the model are serialized from a trained Keras model into a JSON file. Here, this file is loaded by the browser and used to run the neural network in the browser, on-the-fly with the loaded sample images. Five sample images are randomly loaded per button-click below.

Offline, this trained model approached 98.1% test accuracy. Of course, there are vastly better models, such as those based on convolutional neural networks.
